# Initial Concept
A multi-agentic framework for real-time, autonomous freeform metasurface design.

# Product Guide: MetaChat

## Target Users
MetaChat is designed for a diverse range of users in the photonics community:
- **Nanophotonics researchers and designers** seeking to explore complex freeform structures without the computational overhead of traditional solvers.
- **Optical engineers in industry** who require rapid prototyping and optimization for metasurface-based devices.
- **Students** who can use the framework to learn and experiment with metasurface design through an intuitive, interactive interface.

## Primary Goals
The framework aims to revolutionize metasurface design by achieving:
- **Near Real-Time Design Cycles:** Transitioning from days-to-weeks to nearly real-time execution by combining agentic logic with deep learning solvers.
- **Automated Multi-Objective Optimization:** Seamlessly handling complex design requirements, including multi-wavelength and multi-angle performance criteria, through autonomous agent collaboration.

## Core Features
- **Agentic Iterative Monologue (AIM):** A novel system that orchestrates collaboration between multiple specialized agents and human designers, automating the design workflow.
- **FiLM WaveY-Net Solver:** A semi-general fullwave surrogate solver that provides millisecond-speed simulations with high fidelity to Maxwell's equations.
- **Conditional Fullwave Modeling:** Support for variable conditions, including source angle, wavelength, and material properties, enabling flexible and comprehensive design exploration.
- **Web-Based Interactive Interface:** A user-friendly chat interface that allows designers to interact with the agentic system and visualize designs in real-time.
